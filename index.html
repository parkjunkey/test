DDL 리뷰 요청

DDL 구문을 제출하면 DBA 봇이 안전성·영향도·컨벤션을 자동 분석합니다.

리뷰 전용 — 구문 직접 실행 없음

1. DB 정보

대상 데이터베이스 이름을 정확히 입력해 주세요.

2. DDL 구문 *

MySQL / MariaDB / Aurora MySQL은 테이블 및 컬럼에 COMMENT 절을 반드시 포함해야 합니다.

3. 테이블 주석 스크립트 * MS-SQL 필수

MS-SQL은 sp_addextendedproperty 등을 이용한 테이블·컬럼 주석 스크립트를 DDL과 함께 제출해야 합니다.

4. 요청자 정보

✅

요청이 접수되었습니다

DBA 봇이 DDL 구문을 검토 중입니다.
리뷰 결과는 입력하신 연락처로 전달드립니다.

※ 현재는 샘플 페이지입니다. 실제 리뷰는 진행되지 않습니다.
